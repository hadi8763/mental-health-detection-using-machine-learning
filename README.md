# Mental Health Detection using Machine Learning

This project uses machine learning models to detect mental health conditions from social media text data.

## Dataset
- Source: Kaggle & scraped social media
- Format: Two columns (`text`, `label`)
- 27,977 samples

## ML Models Used
- Support Vector Machine (SVM)
- Logistic Regression
- Random Forest
- Naive Bayes

## Accuracy Results
- SVM: 92%
- Logistic Regression: 91%
- Random Forest: 88%
- Naive Bayes: 87%

## Dependencies
See `requirements.txt`

## How to Run
1. Clone the repo
2. Install the dependencies
3. Open `Final.ipynb` and run all cells
